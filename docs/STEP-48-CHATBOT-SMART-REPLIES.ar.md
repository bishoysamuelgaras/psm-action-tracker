# STEP 48 — Chatbot Smart Replies

## ما الذي تغير
- تحسين عرض الردود داخل الشات
- إضافة عنوان + شرح + خطوات + أسئلة مرتبطة داخل نفس الرسالة
- دعم أفضل للأسئلة السريعة والـ related questions عن طريق intentKey مباشر
- توسيع المحتوى الفعلي للبوت في:
  - Action details
  - تعديل الـ Action
  - كتابة Recommendation بشكل أوضح
  - مراجعة/طباعة Reports
  - Logs filters + pagination
  - Settings users/roles help
