# STEP 47 — Chatbot Compact Layout

## الهدف
تصغير حجم الشات وتحسين تموضعه داخل الصفحة حتى لا يظهر جزء منه مقصوص، خصوصًا على الشاشات المتوسطة أو عندما يزيد محتوى الرسائل.

## ما الذي تم تعديله
- تقليل الحجم العام للنافذة.
- ربط ارتفاع نافذة الشات بارتفاع الشاشة.
- إخفاء زر الروبوت أثناء فتح الشات.
- جعل أسئلة البداية في سطر أفقي قابل للتمرير بدل أن تأخذ ارتفاعًا كبيرًا.
- إخفاء quick questions بعد أول رسالة من المستخدم لتوسيع مساحة المحادثة.
- جعل quick actions داخل الرسائل في سطر أفقي قابل للتمرير.
- تحسين توزيع المساحات بين الهيدر ومنطقة الرسائل وحقل الإدخال.

## الملفات المعدلة
- `src/features/chatbot/components/ChatbotDrawer.tsx`
- `src/features/chatbot/components/ChatbotQuickQuestions.tsx`
- `src/features/chatbot/components/ChatbotMessageList.tsx`

## ماذا أفعل؟
1. خذ backup أو اعمل commit.
2. انسخ الملفات إلى نفس المسارات داخل المشروع.
3. شغّل محليًا بواسطة:

```bash
netlify dev
```

4. اختبر على:
- صفحة Recommendations
- صفحة Actions
- شاشة لابتوب متوسطة
- شاشة أصغر أو بتكبير المتصفح

## النتيجة المتوقعة
- الشات يظهر كامل داخل الشاشة.
- لا يظهر زر الروبوت أسفل النافذة أثناء الفتح.
- مساحة الرسائل تصبح أكبر وأوضح.
- الأسئلة المقترحة لا تستهلك ارتفاعًا كبيرًا.
