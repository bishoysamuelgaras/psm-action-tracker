# Step 22 — Source form tune-up

التعديل ده مركز على جزء Step 1 + Source snapshot داخل صفحة Sources.

## اللي اتظبط
- `Source no.` بقى Full width وواضح
- `Source type` و `Source date` بقوا صف مرتب بدل التزاحم
- `Department` بقى تحتهم بعرض كامل
- `Source snapshot` بقى كارت جانبي/سفلي مرتب حسب المساحة
- `Record info` بقى أوضح في وضع التعديل
- إزالة الإحساس بالزحمة في الجزء العلوي من الفورم

## الملفات
- `src/features/sources/components/SourceForm.tsx`

## التنفيذ
انسخ الملف فوق مشروعك بنفس المسار ثم شغّل:

```bash
npm run build
npm run dev
```
