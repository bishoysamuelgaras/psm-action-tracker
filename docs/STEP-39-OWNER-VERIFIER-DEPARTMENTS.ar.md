تم في هذه الخطوة إضافة نفس منطق اختيار الإدارات داخل Action form لكل من:
- Owner
- Verifier

مثل Responsible تمامًا:
- From system
- Enter name

ولو اخترت From system سيظهر لك الآن:
- System users
- Departments

مهم:
- التعديل محافظ جدًا ومحصور في ActionForm فقط.
- لم يتم تغيير أي SQL أو APIs جديدة هنا.
- يفترض أن خطوة departments داخل Responsible كانت مطبقة بالفعل.
