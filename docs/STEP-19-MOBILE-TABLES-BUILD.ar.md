الخطوة دي فيها 3 تحسينات رئيسية:

1) إصلاح build
- تم تضمين ملفات الـ typing و RPC والـ API اللازمة حتى يمر `npm run build` بنجاح.
- تم إصلاح خطأ RecommendationsPage الخاص بـ `Field`.

2) تحسين الـ UI والـ card layout
- تم تبسيط شكل الكروت الإحصائية.
- تم ضبط الاتجاه العام للواجهة ليظهر منظمًا مع النصوص الإنجليزية.
- تم تحسين وضوح النص داخل inputs / selects / textarea.

3) تحسين الجداول على الموبايل
- Sources: قائمة كروت على الموبايل بدل overflow التقليدي.
- Recommendations: تم إصلاح الصفحة مع الحفاظ على عرض الموبايل.
- Actions: قائمة كروت على الموبايل بدل جدول أفقي.
- Filters في Sources و Actions أصبحت أكثر مرونة على الشاشات الصغيرة.

بعد نسخ الملفات:
1. شغّل `npm run build`
2. جرّب Sources / Recommendations / Actions على الموبايل
3. لو أردت، الخطوة التالية تكون تحسينات أعمق على Dashboard و Reports والـ forms نفسها.
